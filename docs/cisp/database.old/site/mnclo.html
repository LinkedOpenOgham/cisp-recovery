Site Notes

RCAHMW/1925, 207: `Temple Druid...The present house was rebuilt for a hunting-box early in the 19th century, when the name was changed to Temple Druid; the former name of Bwlch y clawdd is still used by old inhabitants'.

The stone was found next to the road near this building.
