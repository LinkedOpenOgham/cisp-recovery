Names

  • Vailathi (Language: Goidelic; Gender: male)
    Thomas/1994, 265, `subsequent versions of [this name is]...Fáelath, `Howler', perhaps another word implying `Wolf''.

    Jackson/1953, 566, states that it `is very probable' that the name is Irish, albeit unknown, later giving Faelad.

  • Vrochani (Language: Goidelic; Gender: male)
    Thomas/1994, 265: `subsequent versions of [this name is]... Fróechán, `Little Heather''.

    Jackson/1953, 566, states that the name is unknown, unless coming from Irish Froechan, `as is very probable'.
