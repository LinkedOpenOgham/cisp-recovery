Names

  • Blaimac (Language: Goidelic; Gender: male)
    Petrie/1872, 36: `This is for Blaithmac, possibly the same name as Blathmac, from blath, `a flower', and mac `son'...The name Blathmac was borne by an Abbot of Clonmacnois, whose death is...recorded A.D. 891...and the same event is given in the Chronicon Scottorum, at the year 896.'

    Lionard/1961, 162, accepts this identification.
